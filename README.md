# Seb's Collection of Mathematical Procedures

A comprehensive reference document for **01001 Mathematics 1a (Polytechnical Foundation)** at DTU (Technical University of Denmark). Written in [Typst](https://typst.app/), this collection provides definitions, step-by-step solution methods, and worked examples organized by weekly course topics.

---

## Purpose

- Provide a complete reference of definitions and theorems for the 01001 course
- Document systematic solution procedures for all common problem types
- Include worked exam-style examples with detailed solutions
- Serve as a quick-reference during revision and exam preparation

---

## Contents

The document is structured to follow the 01001 course progression:

### Definitions & Fundamentals

- **Notation og Symboler** — Mathematical notation and symbols
- **Udsagnslogik** — Propositional logic (negation, conjunction, disjunction, implication, biconditional)
- **Mængder** — Sets (subsets, intersection, union, difference)
- **Funktioner** — Functions (injective, surjective, bijective)
- **Komplekse Tal** — Complex numbers
- **Polynomier** — Polynomials
- **Matricer** — Matrices (fundamentals, determinants, rank)
- **Lineære Ligningssystemer** — Systems of linear equations
- **Vektorrum** — Vector spaces
- **Lineære Afbildninger** — Linear transformations
- **Egenværdier og Diagonalisering** — Eigenvalues and diagonalization
- **Differentialligninger** — Differential equations

### Reference Tables

- FOIL method for polynomial multiplication
- Radian/degree conversion tables
- Trigonometric values reference
- CAST rule (signs in quadrants)
- Unit circle diagram
- Characteristic polynomial formulas

### Weekly Topics & Solution Methods

- **Uge 1:** Propositional logic — tautologies, truth tables
- **Uge 2:** Sets and functions — absolute value equations
- **Uge 3-4:** Complex numbers — polar form, roots, arguments
- **Uge 5:** Polynomials and induction — roots, division algorithm, proofs
- **Uge 6:** Linear systems and Gaussian elimination — free variables, invertibility
- **Uge 7:** Matrix algebra and determinants — rank, nullity
- **Uge 8-9:** Vector spaces, basis, coordinates — span, dimension, subspaces
- **Uge 10:** Linear transformations — kernel, image, change of basis
- **Uge 11:** Eigenvalue problems and diagonalization — eigenspaces, similarity
- **Uge 12:** Systems of linear ODEs — real and complex eigenvalues
- **Uge 13:** n-th order differential equations — homogeneous and inhomogeneous solutions

### Exam Practice

- Worked exam problems with full solutions (Opgave 1–6)
- Multiple choice problem types (MC1–MC8)
- Additional solution techniques for common exam questions
